# Cyberpunk System Monitor

Real-time system monitoring dashboard with a cyberpunk aesthetic.

## Quick Start

```bash
python3 system_monitor2.py
```

Opens at `http://127.0.0.1:8888`

## Requirements

- Python 3.6+
- Linux (reads from `/proc` for system metrics)
- No external dependencies

## Features

- Live CPU, memory, disk, and network monitoring
- Real-time charts and visualizations
- Process tracking
- System load averages
- Cyberpunk-themed UI with 3D graphics
